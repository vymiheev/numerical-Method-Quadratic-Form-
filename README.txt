Решает интегральное неоднородное уравнение Фредгольма 2го рода методом квадратур: прямоугольника. 
Для достижение большего порядка аппроксимации O(h^2), узлы берутся внутри интервала. 
Внутри каждого класса содержится его описание.

Дефолтные данные:
	интервал интегрирование [0,1]
	ядро вырожденное: t+xt^2
	лямбда = 1
	f(x)=1;
	вид уравнения: u(x) = J (t+xt^2)u(t) dt + 1
